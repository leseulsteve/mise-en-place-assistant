# Mise en Place Assistant

Do not install this.

This repository is a private Home Assistant experiment, not a supported custom
integration, product, template, or recommendation. It exists for local testing
and personal development only.

It may change without notice, break without warning, and make assumptions that
only fit one private setup. Do not clone it for production, household tracking,
food safety, inventory accuracy, or anything you would be annoyed to debug at
dinner time.
